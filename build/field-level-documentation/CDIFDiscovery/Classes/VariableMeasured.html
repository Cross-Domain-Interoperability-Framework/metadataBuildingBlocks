VariableMeasured

CDIFDiscovery::Classes::VariableMeasured

Definition

schema:variableMeasured / cdifVariableMeasured - description of what the dataset measures (e.g. temperature, pressure). In Discovery this is a flat description; in DataStructure it gains role and structural references.
