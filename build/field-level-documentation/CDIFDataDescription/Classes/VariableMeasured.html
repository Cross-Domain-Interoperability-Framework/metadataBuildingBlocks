VariableMeasured

CDIFDataDescription::Classes::VariableMeasured

Definition

schema:variableMeasured payload at the Data Description level. Wraps a DDI-CDI InstanceVariable reference with additional CDIF-required properties: cdif:physicalDataType and (optionally) cdif:role.
