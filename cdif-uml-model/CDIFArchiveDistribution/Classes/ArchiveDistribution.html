ArchiveDistribution

CDIFArchiveDistribution::Classes::ArchiveDistribution

Definition

A schema:DataDownload distribution that is an archive file (e.g. ZIP, TAR) containing multiple component files described by schema:hasPart. The archive itself is independently accessible (contentUrl); its component files are NOT independently accessible and are typed as schema:MediaObject (not DataDownload). Inherits the standard DataDownload attributes (name, description, contentUrl, contentSize, encodingFormat, conformsTo) from CDIFCore::DataDownload (see that page); this UML restates only the archive-specific surface area.
