ArchivePart

CDIFArchiveDistribution::Classes::ArchivePart

Definition

A component file within an archive distribution. Typed as schema:MediaObject; carries filename, MIME type(s), optional size and checksum, and optional schema:about back-references when the part is a sidecar metadata file.
