CDIFDataDescription::Classes::AbstractMapping

Definition

Abstract supertype for the cdif:hasPhysicalMapping family. Each concrete subtype maps a variable to a specific physical representation: PhysicalMapping (generic index-based), TextMapping (delimited / fixed-width text), LocatorMapping (locator string for structured data). Every concrete mapping carries a cdif:formats_InstanceVariable reference to the variable it describes.
