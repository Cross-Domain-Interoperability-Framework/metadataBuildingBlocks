DataDescriptionWithStructure::Classes::DataDownload

Definition

schema:DataDownload extended for the Data Description profile - adds the character-set attribute, the per-field physical mappings (cdif:hasPhysicalMapping), and a data fingerprint (cdi:fingerprint). File size is carried by the Core schema:contentSize attribute. Co-typed cdi:PhysicalDataSet in instances; TabularTextDataSet and StructuredDataSet specialize it for text-tabular and structured serializations. Composes with the core DataDownload at the model-composition step.
